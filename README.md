# MacroSolver AI

Aplicación fullstack (React + Node.js) para la gestión inteligente de macros y recetas usando Inteligencia Artificial.

## Organización del Proyecto
El desarrollo se gestiona mediante metodología Kanban.
* **Tablero de Trello:** [https://trello.com/b/ldZPBZ40/macrosolver]
